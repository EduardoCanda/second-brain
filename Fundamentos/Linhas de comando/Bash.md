**B**orune **A**gain **Sh**ell (bash) é um interpretador de comandos mais comum em sistemas [[Linux]], assim como [[MacOS]].

Podemos pensar que o shell é uma maneira de conversarmos com o sistema operacional. Ele é um intermediário entre as chamadas de sistemas que fazemos.

Com o bash é possível shell scripts para automação de tarefas repetitivas, tarefas no [[crontab]] e rotinas que façam sentido para o sistema.
