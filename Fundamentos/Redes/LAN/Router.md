O trabalho do router é conectar as redes e passar dados entre elas. A forma utilizada para isso é o *routing*, por isso o nome **router**.

*Routing* é processo de trafegar entre as redes. Isso envolve criar caminho entre elas e só assim os dados poderão serem entregues.
