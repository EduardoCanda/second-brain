**SGBD** (Sistema Gerenciador de Banco de Dados) ou **DBMS** (Database Management System), servem como interface entre o usuário e o DB.

Permitem configurar o DB, inserir, alterar, excluir e obter os dados também.

Alguns exemplos de SGBDs: MySQL, MongoDB, Oracle Database e Maria DB.

A interação entre o usuário e o DB pode ser usada a linguagem SQL.
