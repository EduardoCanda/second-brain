# Docker — Volumes e Persistência

Volumes permitem persistir dados fora do ciclo de vida do container.

## Tipos
- Volume
- Bind mount
- tmpfs

## Características
- Compartilháveis entre containers
- Persistem após remoção do container
- Gerenciados pelo Docker

## Comandos
- docker volume create
- docker volume ls
- docker volume inspect
