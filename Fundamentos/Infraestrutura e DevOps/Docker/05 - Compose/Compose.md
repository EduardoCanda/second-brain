# Docker Compose

Docker Compose permite definir e executar múltiplos containers com um único arquivo YAML.

## Conceitos
- Serviços
- Networks
- Volumes
- Dependências

## Arquivo
- docker-compose.yml

## Comandos
- docker compose up
- docker compose down
- docker compose logs
- docker compose ps
