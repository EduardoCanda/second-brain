No gerenciamento de usuários é possível cadastrar todos os dados que fizerem sentido para ele. Por exemplo, nome completo, endereço, e-mail, telefone, cargo da empresa, setor, entre outras coisas mais.

Criação de usuários, edição e remoção dos mesmos também.

## Delegação:
**Delegation** ou **Delegação (pt-br)** é o processo de garatir privilégios a usuários do servidor. É possível garantir a uma pessoa de helpdesk, o poder de resetar a senha de **OUs** de toda a empresa por exemplo.
