Além dos dispositivos reais é possível montar um sistema de arquivos virtual, existe uma pagina dedicada para esse assunto [[Sistema de Arquivos Virtual(VFS)|aqui]], porém a implementação do kernel permite que para o sistema seja transparente esse aspecto, sendo assim conseguimos ler/escrever nesse sistema como em qualquer outro dispositivo de bloco convencional.
