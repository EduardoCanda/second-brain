Na descompactação de arquivos utilizamos a logica reversa da [[Compactacão|compactação]], essa técnica é necessário a partir de um momento que há uma necessidade de recuperar os arquivos para determinada finalidade como por exemplo visualizar fotos antigas, analisar logs antigos de aplicações ou após uma [[Transferencia de Dados|transferência de dados]] ter sido executada com sucesso, uma das ferramentas que usamos para essa finalidade é o [[tar|tar]] ele apresenta diversas opções que interessantes para essa finalidade, para entender mais a respeito acesse essa [[tar#Exemplo de compressão, descompressão e visualização|parte]].
