O Comando retorna a lista de redes disponíveis, ele é um alias para o comando [[nmcli device wifi list]], é possível visualizar a lista de redes wifi disponíveis em uma visão simplificada.
# Exemplos

Listando status de dispositivos
```bash
nmcli device wifi wlp4s0
```
