O Comando é utilizado para monitorar alterações em uma conexão, sempre que há alguma modificação na conexão o comando imprime um alerta no terminal notificando que a conexão foi modificada.
# Exemplos

Monitorando uma conexão
```bash
nmcli connection monitor Bertie
```
