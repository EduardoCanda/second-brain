O Comando ip address delete é extremamente simples, com ele é possível apagar configurações de endereçamento de uma interface de uma forma muito prática.
# Exemplos

Também é possível remover um endereço de ip atrelado a uma interface de rede através do comand **`del`**, 
```bash
ip address delete 192.168.15.50/24 dev enp5s0
```
