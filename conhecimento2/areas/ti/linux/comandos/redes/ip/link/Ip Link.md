O Comando ip link é muito utilizado na gestão de interfaces de redes, diferente do [[ip-address]] que gerenciar as configurações do [[protocolo-enderecamento-ipv4]], podemos realizar diversas operações em interfaces de redes fisicas e virtuais.

Podemos associar interfaces a [[Namespaces]] também, podendo trabalhar muito bem com ambientes isolados([[Namespaces]]), temos diversos metodos que podemos chamar, cada metodo irá realizar uma ação no contexto de interfaces de redes.
