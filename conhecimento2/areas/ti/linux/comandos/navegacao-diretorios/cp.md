O Comando copy copia um arquivo/diretório, pode ser tanto uma copia simples como uma copia recursiva


## **Exemplos**

Copiando um arquivo

```
cp arquivoAlvo nomeCopia
```
Copiando um diretório com todo seu conteudo
```
cp -r diretorioAlvo nomeCopia 
```
Manual do comando
```bash
man cp
```
