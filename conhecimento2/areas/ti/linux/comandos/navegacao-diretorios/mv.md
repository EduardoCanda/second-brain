O Comando move pode ser utilizado tanto para mover um arquivo, quanto para renomear o mesmo, a forma de usar ele é simples como no exemplo abaixo:

## **Exemplo**

Renomeando arquivos
```bash
mv nomeAtualArquivo novoNomeArquivo
```

Movendo arquivo para outro diretório
```bash
mv nomeArquivo diretorioDestino
```

Movendo varios arquivos para o mesmo diretório
```bash
mv arquivo1.txt arquivo2.txt arquivo3.txt arquivo4.txt diretorio
```

Manual do comando
```bash
man mv
```
