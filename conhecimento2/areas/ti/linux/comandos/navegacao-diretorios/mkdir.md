O comando é utilizado para criar diretórios, sua utilização é bem simples como no exemplo a baixo.

## **Exemplos**

Criar o diretório especificado
```bash
mkdir diretorios...
```
Criar o diretório especificado e sub diretórios
```bash
mkdir -r diretorio/sub/dire/torio...
```
Manual do comando
```bash
man mkdir
```
