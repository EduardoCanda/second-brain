O Comando rm pode ser utilizado para apagar arquivos/diretórios.

## **Exemplos**

 Apagar um arquivo
```bash
rm arquivos....
```

 Apagar o diretório vazio
```bash
rm -d diretorios....
```


 Apagar o diretório e seus arquivos/diretórios internos
```bash
rm -r diretorio....
```

Manual do comando
```bash
man rm
```
