O Comando cd(Change Directory) é utilizado para troca de diretórios no [[Sistema de arquivos|Sistema de arquivos]], é possível referenciar a pasta do usuário atravez do indicador `~`, ou somente `cd`.

**Exemplos:**

Acessando um diretório
```bash
cd teste
```

**Saida:**
![[cd.png]]

Voltando para o diretório anterior
```bash
cd ..
```

Acessando home:
```bash
cd
```
