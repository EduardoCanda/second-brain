O IAM é um serviço da AWS de gerenciamento de acessos, é possível controlar a específicidade que uma entidade terá dentro de nossa cloud e também específicar algumas configurações de segurança como por exemplo MFA e politica de senhas forçando requisitos minimos para tal.

Podemos criar usuários, grupos e também roles, cada qual poderá ter previlégios específicos a depender da organização desejada.
