Quando pensamos em Java uma das suas principáis features sem dúvidas são as [[multi-threading-java|Threads]], elas são consideradas unidades minimas em um processo em execução e com elas podemos executar processos com paralelismo e assincronismo.

Uma das melhores práticas é utilizar [[variaveis-imutaveis-java|Variaveis Imutaveis]], para proteger nosso software de eventuais problemas de [[multi-threading-java|race condition]] por exemplo, e atravéz dessa abordagem podemos entender a respeito dos termos [[thread-safe-non-safe|Thread Safe X Thread Non Safe]] que de uma forma simplificada é como podemos trabalhar com contextos multithreading com com uma certa segurança.
