Durante o debugging é possível usar ferramentas que aumentem sua produtividade, uma das principais é o Visual Code, com ele podemos instalar o Plugin do [[kubernetes]] para conectar o mesmo ao nosso Cluster e com isso facilitar os comandos de logs, describe, e até listagem de [[custom-resources|CRDs]] do [[introducao-argocd|Argo CD]], com isso temos um ganho de performance além do aprendizado que é exponencial.
