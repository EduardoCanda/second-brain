# 🧱 Índice — Fundamentos

A pasta **Fundamentos** existe para consolidar a base técnica que sustenta todos os outros tópicos do Second Brain.

## Objetivo

- Construir entendimento sólido de conceitos essenciais.
- Priorizar explicações didáticas antes de ferramentas específicas.
- Servir como pré-requisito para os guias e aprofundamentos.

## O que você encontra aqui

- Redes
- Sistemas operacionais
- Programação
- Banco de dados

## Quando usar esta seção

Use esta pasta quando estiver começando um tema novo ou quando precisar revisar conceitos centrais para destravar estudos mais avançados.
