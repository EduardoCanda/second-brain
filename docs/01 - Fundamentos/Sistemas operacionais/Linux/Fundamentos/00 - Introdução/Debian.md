# Debian

**Debian** é uma distribuição Linux conhecida por estabilidade e compromisso com software livre. É amplamente utilizada como base para outras distribuições, como Ubuntu.

## Por que é relevante
- Ciclos de atualização mais conservadores.
- Repositórios amplos de pacotes.
- Muito usado em servidores e ambientes críticos.
