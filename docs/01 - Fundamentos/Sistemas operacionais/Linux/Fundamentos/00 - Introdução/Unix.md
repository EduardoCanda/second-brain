# Unix

**Unix** é uma família de sistemas operacionais que influenciou fortemente a computação moderna. Seus princípios de modularidade, portabilidade e ferramentas pequenas inspiraram sistemas como Linux e macOS.

## Conceitos importantes
- Arquitetura multiusuário e multitarefa.
- Forte uso de terminal e scripts.
- Filosofia de compor programas simples para resolver tarefas complexas.
