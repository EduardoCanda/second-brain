# Ubuntu

**Ubuntu** é uma distribuição Linux baseada em Debian, popular tanto em desktops quanto em servidores. É conhecida por facilidade de uso e boa documentação.

## Pontos fortes
- Instalação simples para iniciantes.
- Comunidade ativa e vasta documentação.
- Bom suporte para desenvolvimento e cloud.
