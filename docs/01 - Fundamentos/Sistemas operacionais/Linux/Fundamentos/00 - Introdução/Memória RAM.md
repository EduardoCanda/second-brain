# Memória RAM

A **RAM (Random Access Memory)** é a memória volátil usada para armazenar dados temporários dos programas em execução. Quanto maior a RAM disponível, maior a capacidade de multitarefa do sistema.

## Resumo prático
- É rápida, mas perde dados ao desligar o computador.
- Impacta desempenho de aplicações e sistema operacional.
- Trabalha junto com CPU e armazenamento.
