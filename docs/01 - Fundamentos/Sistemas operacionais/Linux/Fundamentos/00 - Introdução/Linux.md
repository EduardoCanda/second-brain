Assim como [Windows](../../../Windows.md), [iOS](iOS.md) e [MacOS](../../../../Sistemas operacionais/MacOS.md), o **Linux** é um sistema operacional dos mais utilizados do mundo. Ele equipa carros inteligentes, dispositivos android, supercomputadores, servidores enterprise, websites, painéis de controle e muito mais.

---
## Variações do Linux:
O nome "**Linux**" atualmente é um termo utilizado para múltiplos sistemas operacionais que são baseados em [Unix](Unix.md). Graças ao Linux, a comunidade [Open Source](Open Source.md) ganhou força e com isso eles mesmo desenvolveram várias versões de Linux.

Há variações das mais diversas. Uma para cada propósito. As versões mais comuns são as baseadas em [Ubuntu](Ubuntu.md) e [Debian](Debian.md) por serem muito flexíveis.

Por exemplo, um Ubuntu pode servir como base para um servidor web, uma API, websites or um computador totalmente básico.
*Nota: Um servidor Ubuntu pode rodar em um computador com 512MB de [Memória RAM](Memória RAM.md)*
