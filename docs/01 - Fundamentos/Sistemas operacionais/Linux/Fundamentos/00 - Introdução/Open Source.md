# Open Source

**Open Source** (código aberto) é um modelo no qual o código-fonte de um software pode ser estudado, modificado e redistribuído.

## Benefícios
- Transparência e auditoria de segurança.
- Evolução colaborativa pela comunidade.
- Menor dependência de um único fornecedor.
