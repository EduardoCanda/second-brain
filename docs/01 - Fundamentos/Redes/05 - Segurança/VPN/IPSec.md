# IPSec

**IPSec (Internet Protocol Security)** é um conjunto de protocolos para proteger comunicação em nível de IP, oferecendo autenticação, integridade e criptografia.

## Componentes comuns
- AH e ESP para proteção dos pacotes.
- Modo transporte e modo túnel.
- Uso de IKE para negociação de chaves.
