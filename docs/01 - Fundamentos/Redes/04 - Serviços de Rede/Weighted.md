# Weighted

No contexto de balanceamento de carga, **Weighted** significa distribuir requisições com pesos diferentes entre servidores, privilegiando os que têm maior capacidade.

## Exemplo
Se um servidor tem peso 2 e outro peso 1, o primeiro tende a receber aproximadamente o dobro das requisições.
