**B**orune **A**gain **Sh**ell (bash) é um interpretador de comandos mais comum em sistemas [Linux](../Sistemas operacionais/Linux/Fundamentos/00 - Introdução/Linux.md), assim como [MacOS](../Sistemas operacionais/MacOS.md).

Podemos pensar que o shell é uma maneira de conversarmos com o sistema operacional. Ele é um intermediário entre as chamadas de sistemas que fazemos.

Com o bash é possível shell scripts para automação de tarefas repetitivas, tarefas no [crontab](crontab.md) e rotinas que façam sentido para o sistema.
