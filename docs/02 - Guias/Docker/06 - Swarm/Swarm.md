# Docker Swarm

Docker Swarm é o modo de orquestração nativo do Docker.

## Conceitos
- Cluster
- Manager
- Worker
- Services
- Desired State

## Recursos
- Load balancing interno
- Failover automático
- Escalabilidade

## Comandos
- docker swarm init
- docker node ls
- docker service create
- docker service scale
