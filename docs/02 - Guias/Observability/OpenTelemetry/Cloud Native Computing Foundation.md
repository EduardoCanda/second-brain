# Cloud Native Computing Foundation

A **Cloud Native Computing Foundation (CNCF)** é uma fundação que hospeda e governa projetos essenciais do ecossistema cloud native, como Kubernetes, Prometheus e OpenTelemetry.

## Papel no ecossistema
- Mantém governança neutra para projetos abertos.
- Incentiva padronização e interoperabilidade.
- Apoia comunidades técnicas e eventos globais.
