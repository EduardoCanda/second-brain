# Docker — Segurança

Segurança em Docker envolve imagens, containers e runtime.

## Conceitos
- Docker Content Trust
- Image signing
- Capabilities
- Seccomp
- AppArmor
- SELinux

## Boas práticas
- Não usar root
- Imagens mínimas
- Atualizações frequentes
- Scan de vulnerabilidades
