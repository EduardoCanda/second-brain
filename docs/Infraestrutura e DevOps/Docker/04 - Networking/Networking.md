# Docker — Networking

Docker fornece redes virtuais para comunicação entre containers.

## Tipos de rede
- bridge (default)
- host
- none
- overlay
- macvlan

## Conceitos importantes
- Port mapping
- DNS interno
- Service discovery
- Isolamento de tráfego

## Comandos
- docker network ls
- docker network create
- docker network inspect
