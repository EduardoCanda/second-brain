# Docker — Troubleshooting

Conjunto de técnicas para diagnóstico de problemas em containers.

## Ferramentas
- docker logs
- docker inspect
- docker events
- docker stats

## Problemas comuns
- Container não inicia
- Porta não exposta
- Volume não persiste
- Rede não resolve DNS
