Container é uma instância em execução de uma imagem Docker.
## Características
- Processo isolado
- Efêmero por padrão
- Compartilha o kernel do host

## Ciclo de vida
- created
- running
- paused
- stopped
- removed

## Comandos comuns
- docker run
- docker ps
- docker exec
- docker logs
- docker stop
- docker rm
