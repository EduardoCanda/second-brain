# Lab: Refatoração de Nota

Descrição curta: exercício para converter uma nota monolítica em quatro páginas especializadas.

## Sumário rápido
- [Cenário](#cenario)
- [Objetivo](#objetivo)
- [Passos](#passos)
- [Critérios de aceite](#criterios-de-aceite)
- [Ver também](#ver-tambem)

## Cenário
Você possui uma nota única com teoria, comandos, erros e recomendações misturados.

## Objetivo
Gerar páginas separadas por tipo:
- Getting Started
- Reference
- Troubleshooting
- Best Practices

## Passos
1. Extraia seções por intenção.
2. Aplique templates padrão.
3. Conecte com “Ver também”.
4. Revise tags e nomenclatura.

## Critérios de aceite
- Quatro páginas independentes.
- Cada uma com mini sumário.
- Links cruzados funcionando.
