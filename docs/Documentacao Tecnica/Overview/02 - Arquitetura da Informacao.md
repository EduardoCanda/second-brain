## Princípio de organização
Cada página pertence a um tipo:
- `#concept` para fundamentos
- `#tutorial` para guias de execução
- `#reference` para consulta rápida
- `#operations` para playbooks

## Mapa de seções
- Overview → visão macro
- Fundamentals → bases e templates
- Guides → execução passo a passo
- Deep Dives → análise aprofundada
- Labs → prática dirigida
- Playbooks → operação recorrente
