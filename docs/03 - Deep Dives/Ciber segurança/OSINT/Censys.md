# Censys

**Censys** é uma plataforma de coleta e busca de dados de hosts e certificados na internet, muito usada para mapeamento de exposição externa.

## Aplicações comuns
- Inventário de ativos expostos.
- Busca por certificados TLS e domínios associados.
- Monitoramento de mudanças de postura de segurança.
