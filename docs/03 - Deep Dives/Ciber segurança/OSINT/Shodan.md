# Shodan

O **Shodan** é um mecanismo de busca para dispositivos e serviços expostos na internet (câmeras, roteadores, servidores, ICS, etc.).

## Uso em segurança
- Descoberta de superfície de ataque.
- Identificação de serviços mal configurados.
- Apoio em reconhecimento para testes autorizados.
