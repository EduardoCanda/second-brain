# Have I Been Pwned

**Have I Been Pwned** é um serviço que permite verificar se e-mails ou senhas apareceram em vazamentos de dados conhecidos.

## Boas práticas
- Verificar contas corporativas periodicamente.
- Trocar senhas comprometidas.
- Ativar MFA sempre que possível.
