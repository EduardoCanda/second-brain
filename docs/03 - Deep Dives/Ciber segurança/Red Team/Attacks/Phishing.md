# Phishing

**Phishing** é um ataque de engenharia social no qual o atacante engana a vítima para obter credenciais, dados financeiros ou executar ações maliciosas.

## Técnicas comuns
- E-mails ou mensagens que imitam empresas confiáveis.
- Páginas falsas de login.
- Sensação de urgência para induzir erro humano.
