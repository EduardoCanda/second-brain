A ideia principal por trás do **WIndows Domain** é centralizar a administração de componentes comuns entre computadores Windows na rede em um único repositório, chamado **Active Directory (AD)**.

O servidor que hospeda o AD também é conhecido como **Domain Controller (DC)**

![[Domain Controller.png]]

## Vantagens:
- **Centralisar o gerenciamento de identidade:** Todos os usuários em torno da rede podem ser configurados no AD com um esforço mínimo.

- **Gerenciar as políticas de segurança:** Configurando direto do AD e aplicando a todos os usuários e computadores que fizerem sentido.
