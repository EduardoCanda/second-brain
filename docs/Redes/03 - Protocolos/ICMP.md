ICMP - Internet Control Message Protocol

Este protocolo é integrante ao protocolo [[IP]], definido pelo RFC 792. Ele
permite gerenciar informações relacionadas a erros de comunicação entre máquinas. Este protocolo não corrige os erros, no entanto avisa as camadas vizinhas sobre eles.
Portanto, o ICMP é utilizado por todos os roteadores para assinalar um erro de Delivery Problem.
