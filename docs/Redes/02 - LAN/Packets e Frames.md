Pacotes(packets) e frames são pedaços de dados que quando juntados podem fazer um pedaço maior de informação ou mensagem.

Pacote (packet) é um pedaço de dados que pertence a camada 3 Network do **Modelo OSI**, contendo informações como endereço **IP** e payload.

O frame também é um pedaço de dados, pertence a camada 2 Data Link do **Modelo OSI**, e que encapsula o packet e adicionalmente acrescenta o **MAC address**.

Fazendo uma analogia com o mundo real, podemos dizer que o frame é um envelope e o pacote é a carta dentro dele. Portanto o envelope será enviado pelo correio.
